# Project 27: Expanding the ELIXIR TES Network with Life-Science Services

## Abstract

The ELIXIR On-Cloud TES network provides a functional GA4GH TES environment for federated task execution across ELIXIR Nodes. Building on this foundation, we propose a BioHackathon project that aims to leverage the TES network to deploy research services, similar to what has been done for the BioHackCloud (https://biohack.cloud/services), but via TES requests. Such TES-deployed services could support progress in other ELIXIR initiatives, including compute provenance efforts such as ELIXIR-STEERS, secrets management (e.g. OpenBao), and other service integrations such as Galaxy workflow execution in the TES network. More broadly, the project aims to show how the TES network can serve as an execution layer for research services.
 
 The resulting services will be linked and documented at the BioHackCloud and thus made available to the BioHack community. This project directly operationalises the ELIXIR On-Cloud roadmap by using TES as a common interoperability layer not only for task submission but also for interactive life-science analysis services. It contributes a concrete, user-facing use case to the established TES ecosystem, documentation, and best-practice portfolio.
 
 The minimal viable deliverable will demonstrate a complete execution path: user request > TES task dispatch via proTES > service/task execution on a federated node > interactive result delivery.
