# Project 11: VERDANT: Vocabulary Enrichment & Resolution of Diversity Annotations via Neural Technology

## Abstract

Biodiversity and microbiome research increasingly depends on knowledge graphs (KGs) to integrate heterogeneous data across genomic, environmental, and ecological domains. Yet populating these KGs with high-quality semantic links to external ontologies remains a manual, error-prone bottleneck. Existing tools handle ontology construction or querying, but none provide an end-to-end, AI-driven pipeline that automatically scrapes authoritative term sets, builds formal OWL 2 ontologies, and generates confidence-scored SKOS mappings to multiple knowledge bases simultaneously. VERDANT addresses this gap with a production-ready, reusable tool that transforms any structured vocabulary into a FAIR, interlinked ontology layer. The pipeline uses SentenceTransformer embeddings with FAISS vector indexing to produce semantic mappings (exactMatch, closeMatch, relatedMatch) to Wikidata, DBpedia, ENVO, NCBITaxon, and other targets. An expert pre-check interface enables domain specialists to validate AI suggestions, with their decisions feeding back as training data to iteratively improve the classifier. At BioHackathon 2026, we will demonstrate VERDANT on the Natura 2000 protected-area vocabulary (1,955 EU-mandated biodiversity terms, 14,500+ mappings already generated) while generalising the tool for reuse by other communities. Concretely, we will deploy a federated SPARQL endpoint, build API connectors to BioPortal, TeSS, and GBIF, prototype the expert validation and AI fine-tuning infrastructure, and validate cross-domain applicability by enriching a subset of the ELIXIR Microbiome Community's Knowledge Graph. All outputs are MIT-licensed and containerised with Docker Compose.
