# Project 13: ELIXIR AI Ecosystem Integration: Bridging DOME, BioAIrepo and Galaxy

## Abstract

The adoption of machine learning in the life sciences has been transformative, solving landmark challenges and accelerating research. However, researchers face an ongoing reproducibility and reusability crisis. While foundational resources like the DOME Registry for method disclosure and the BioAIrepo for dataset and model deposition exist, a critical functional gap remains between identifying an AI asset and successfully executing it.
 
 This project aims to bridge that gap by functionally integrating the ELIXIR AI ecosystem. Firstly, we will establish a scalable proof-of-concept mechanism to run models directly from the DOME Registry and BioAIrepo via Galaxy. By landscaping existing BioModels ML mechanisms, we will implement front-end features enabling seamless model execution. Secondly, we will jointly enhance interoperability between these resources. Leveraging indexed assets from EBI Search federation, we will successfully pull in and cross-link related software assets from Europe PMC and bio.tools to their corresponding method and data entries in the DOME Registry and BioAIrepo.
 
 Finally, to support researchers in navigating this unified landscape, we will develop improved unified submission documentation and user guidance. This content will explicitly bridge the transition from asset identification to execution. It will be published as a dedicated RDMkit page to ensure optimal viewership and will align with established DOME and OSAI guidelines.
 
 Ultimately, this project will streamline AI model discovery and execution, ensuring ELIXIR infrastructures provide highly actionable, cross-linked resources for the scientific community.
