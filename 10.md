# Project 10: Enabling workflows and EDAM annotations to display in community website deployments

## Abstract

The ELIXIR Toolkit Theme (ETT) enables communities to deploy websites via GitHub, facilitating broad contributions and collaborations, and streamlining the process of building and populating the site. The popularity of the ETT is demonstrated by numerous deployments, including RDMkit, RSQkit, and WorkflowHub documentation. These deployments can also centralise sharing of digital resources such as workflows intended for reuse by communities. This use case requires a streamlined way to find, view, and understand workflows, in the context of an ETT deployed website. 
 
 Whilst ETT already offers the possibility of annotating pages with identifiers from FAIRsharing and bio.tools, support for workflows and ontologies is not implemented. This project will extend the ETT to enable 1) automation of workflow imports from WorkflowHub, 2) creation of interactive tables to present the communities workflows, and 3) automated addition of EDAM ontology annotations at both the workflow table and webpage (i.e. schema.org) level. The latter will build on the existing integration with bio.tools, offer the possibility of manual curation, and increase the findability of these webpages. Moreover, ontology terms (e.g. EDAM) are already used to annotate the bio.tools elements already included on many ETT instances, including RDMkit.
 
 Workflow developers from any community can thus continue using WorkflowHub to make their workflows FAIR, and community coordinators can independently and automatically import these to an ETT deployment along with EDAM annotations. This support would be beneficial for increasing the findability of the pages and enabling better integration across multiple instances of ETT.
