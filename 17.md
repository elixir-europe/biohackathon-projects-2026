# Project 17: Bridging Literature and Practice: Tool Co-citation Networks from Publications and Registered Workflows

## Abstract

InSoLiTo is a graph-based web application that visualizes co-citation networks of bioinformatics tools derived from scientific publications available through the imported tools in the OpenEBench (OEB) collection. However, while it captures how tools are cited together, it does not reflect how they are combined in real-world workflows. This information is available in WorkflowHub, a registry of computational workflows with explicit tool compositions mapped to bio.tools identifiers.
 
 
 This project proposes to bridge these two complementary sources through two interconnected tracks. First, we will fetch workflows metadata from WorkflowHub's API, map its composed tools to their bio.tools IDs, identify those not yet registered in OEB and incorporate them into the platform, and introduce workflows themselves as a new entity type in OEB, enabling explicit connections between tools, workflows, and benchmarking activities. Second, we will extend InSoLiTo's graph interface to represent workflows as directed graphs, making it possible to visualize not only which tools are co-cited, but how they connect and in what order, a dimension currently absent from InSoLiTo, allowing users to explore execution paths and better understand tools interoperability in practice.
 
 Together, these tracks will lay the groundwork for a unified, multi-source view of bioinformatics tool usage, bridging scientific literature, real-world practice, and research infrastructures.
