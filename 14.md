# Project 14: Unlocking Animal Data for Reproducible Research with ARC RO–Crate

## Abstract

This project aims to bridge critical gaps in the implementation of the ISA (Investigation, Study, Assay) model for animal phenotyping by advancing its practical usability. Although last year’s hackathon project confirmed the ISA model’s suitability for animal phenotyping, its adoption remains hindered by the lack of user-friendly tools and incomplete implementation. 
 
 The Plant Science Community has already developed, within the NFDI consortium DataPLANT, a RO-Crate-based Annotated Research Context (ARC) ecosystem that fosters phenotypic ISA models for plants. The re-use of these structures not only enables the efficient implementation of a usable standard for the animal data community but also represents a first step towards an extended data format in the agricultural sector.
 
 The core objective of the project is to set up an ARC for animals with its associated software, like the ARCitect or ISA Wizard, to support intuitive, standardized annotation of ISA-aligned animal phenotypic data. In this process, the new animal-related terms from the ISA checklist need to be integrated or adapted into the software. Therefore, a pilot dataset of animal phenotypic data will be generated and packaged as an ARC, demonstrating the end-to-end workflow from data collection to FAIR-compliant curation. The outcome will be a validated, accessible software tool suite that enables researchers to easily annotate, manage, and share animal phenotyping data in alignment with FAIR principles. This work will significantly enhance reproducibility, data integration, and cross-study comparisons in animal research, laying the groundwork for broader adoption of the ISA model in the life sciences.
