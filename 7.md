# Project 7: PhyloAI: Extending a Machine Learning-Based Quartet Topology Classifier for Phylogenomic-Scale Tree Reconstruction

## Abstract

Phylogenetic inference is a central tool in biology and bioinformatics. Current methods (most commonly used are maximum likelihood and Bayesian inference) face computational challenges due to the ever-growing size of genomic datasets. To address these limitations, we developed PhyloAI, a machine learning method trained on alignment site pattern frequency distributions that reconstructs phylogenetic trees with high accuracy while being up to five orders of magnitude faster than maximum likelihood methods. PhyloAI takes a multiple sequence alignment as an input, predicts the tree topology for every quartet in the alignment using the existing pre-trained neural networks and amalgamates quartet trees with ASTRAL/wQFM to reconstruct a larger a N-taxon phylogenetic tree. In this BioHackathon project, we will extend our working version of PhyloAI by introducing explicit filtering of unreliable quartets, using either reconstructibility predictor networks or softmax-based confidence estimates. By reducing the impact of incorrectly inferred quartets, this step is expected to substantially improve overall tree accuracy. Finally, we aim to make PhyloAI broadly accessible by developing a fully packaged and user-friendly software distribution available via pip and conda.
