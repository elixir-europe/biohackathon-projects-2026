# Project 1: Bridging ELIXIR nodes for collaborative and scalable BYOC in Galaxy and TREs

## Abstract

This project builds on the capabilities of the de.NBI Cloud, extending its SimpleVM infrastructure to support advanced Bring Your Own Compute (BYOC) for Galaxy. By integrating automated infrastructure provisioning with scalable Pulsar execution backends, we aim to enable seamless integration of dedicated compute resources to Galaxy. Building on prior work from EOSC EuroScienceGateway, the project introduces a unified approach that supports both fully managed and user-managed scenarios. In the fully managed model, infrastructure components such as Pulsar, SLURM, and storage are automatically provisioned and integrated. In parallel, user-managed setups are facilitated through automation via Galaxy APIs and CI pipelines, providing flexibility while maintaining reproducibility and ease of use.
 
 Beyond its implementation within de.NBI Cloud, this framework is designed to be utilized by Galaxy providers across ELIXIR Nodes and cloud providers. By standardizing the integration of external compute resources, it bridges distributed infrastructures and connects ELIXIR Nodes, enabling them to collaboratively offer scalable BYOC capabilities and shared access to heterogeneous compute environments.
 
 In parallel, the project will investigate how such infrastructures can support secure processing environments (TRE/SPE) and interoperate with existing solutions such as SURF's Tinker SANE. By leveraging shared technologies (e.g., web-based remote access, reproducible VM images), we aim to enable portable, secure, and user-specific virtual research environments.
 
 The outcome will be a framework that simplifies the onboarding of external compute resources into Galaxy, improves scalability, supports sensitive data use cases, and fosters collaboration across ELIXIR Nodes and cloud providers.
