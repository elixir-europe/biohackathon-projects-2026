# Project 2: Enhancing AI-readiness of multi-modal data submission: a reference workflow

## Abstract

This project will produce a fully documented, end-to-end reference workflow for multi-modal data submission using the MARS approach, to be published as an article and FAIR outputs. It builds on an existing operational prototype pipeline that integrates ISA-JSON for structured metadata exchange with data transfer and orchestration mechanisms, including dedicated test/beta programmatic submission endpoints provided by ELIXIR Deposition Database (EDD).
 
 The workflow demonstrates how heterogeneous life science datasets can be submitted in a reproducible and programmatic manner, combining metadata handling, data dispatch, and large-scale transfer within a single coherent process. The main goal is not to generate new datasets, but to systematically execute and document this workflow to enable reuse and adoption across ELIXIR Nodes.
 
 A key outcome is a clear, step-by-step reference implementation that improves dataset interlinking and supports more consistent metadata structures, thereby increasing the AI-readiness of deposited datasets compared to current heterogeneous submission practices. The project will also identify current gaps in automation, interoperability, and metadata harmonisation, providing concrete recommendations for future improvements.
 
 All materials, including code, documentation, and structured metadata examples, will be released in a reusable and FAIR-compliant manner. The resulting publication will serve as a practical blueprint for Node-based implementations and contribute to the development of a Multi-Node Toolbox for interoperable multi-omics data submission workflows.
